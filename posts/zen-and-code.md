深夜，关掉多余的标签页，只留下编辑器与一段需要解决的逻辑。那一刻的专注，与坐禅时的「止观」竟有几分相似。

## 一次只做一件事

禅修讲究「活在当下」。写代码也是——当我试图同时兼顾太多事情时，bug 往往悄然滋生。把一个大问题拆成一个个小步骤，一次只专注其中之一，反而走得更快、更稳。

## 简洁是一种修行

```js
// 与其堆砌聪明的技巧
const r = a ? (b ? x : y) : (c ? z : w);

// 不如写得让半年后的自己也能读懂
function pick(a, b, c) {
  if (a) return b ? x : y;
  return c ? z : w;
}
```

可读性高于炫技。能被轻松理解的代码，才是真正经得起时间的代码。

## 接受不完美

程序总会有 bug，正如人生总有遗憾。重要的不是追求一次写对，而是建立能持续改进的节奏——测试、重构、反思，循环往复。

> 行到水穷处，坐看云起时。

愿你我都能在代码与生活之间，找到属于自己的那份宁静。
